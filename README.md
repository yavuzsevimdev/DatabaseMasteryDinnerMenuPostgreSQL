# 🍽️ Database Mastery Dinner Menu (PostgreSQL)

## 📖 Project Overview

This project was developed by following the **Database Mastery: ASP.NET Core ile MongoDB-PostgreSQL-MSSQL** course instructed by **Murat Yücedağ**.

The primary objective of this project is to strengthen my knowledge of **ASP.NET Core MVC**, **Entity Framework Core**, and **PostgreSQL** by developing a modern restaurant management application.

The project consists of a user-friendly restaurant website and a comprehensive administration panel where restaurant content can be managed through CRUD operations. Throughout this project, I gained practical experience in dashboard development, statistics reporting, chart visualization, View Components, and PostgreSQL database integration.

## 🏛️ Project Architecture

The project is developed using a layered architecture to improve maintainability, readability, and scalability.

### Project Structure

- 📁 Controllers
- 📁 Models
- 📁 DTOs
- 📁 View Components
- 📁 Views
- 📁 Entity Framework Core
- 📁 PostgreSQL Database

### Design Principles

- MVC (Model-View-Controller) Architecture
- Entity Framework Core Code First
- Dependency Injection
- Repository-based data access
- Responsive UI Design

## 🗄️ Database

This project uses **PostgreSQL** as the primary database management system.

The database was designed using the **Code First** approach with **Entity Framework Core**, allowing tables and relationships to be generated through migrations.

### Database Tables

- Categories
- Products
- Reservations
- Reviews
- Contacts
- Features
- Social Medias
- Chefs
- Abouts
- Address
- OpenHours
- Admins

### Database Tools

- PostgreSQL
- pgAdmin 4
- Entity Framework Core
- Npgsql
- Code First Migrations
